# Liyuu-UTAU-Voicebank
UTAU Chinese voicebank of Liyuu based on live talks (experimental)  

Liyuu UTAU中文音库，截取自直播录音合集。  
目前的版本并非成品，有相当一部分原音设定需要补充完善。  
使用时请遵守当地法律法规。
